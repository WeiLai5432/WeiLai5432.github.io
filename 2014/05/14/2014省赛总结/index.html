2014省赛总结

发表于 2014-05-14 | 分类于 日常 | | 阅读次数 
关于acm我这学期才开始正式学练。距离省赛只有两个月了。这两个月的练习还算密集，基本了解了要学习的内容。省赛的准备我做的也不是很充分，好多算法还不能完全实现，去之前也是害怕做不好。到了威海也就放松下来了，没什么心理压力。
比赛开始前，我们队商量好拿到题分开读，我读DEF题，开始感觉D题可以做，但是好多单词不认识，于是翻字典查。可是这时我发现已经插了好几个气球了，然后刷了board，发现E题阶乘大水，于是任凯写出E题a了，然后我还在读D题。好久才大体弄懂题意，大概是说施放魔法技能伤害什么的，然后我感觉这题可以写，于是就写了，测了下数据也对，交了却wa了，然后我接着读，感觉题目不难，自己找了几组数也都对，跟他们俩讨论了下感觉没错，于是没办法放下了。
然后卢景月开始做F题，我在读其他题，感觉题目难度不是很大，数学的题偏多。这时候F题写出来交了然后过了，我们挺受鼓舞的，他们开始做B，我在看C，感觉能做，但是一直没找到方法。然后我们仨开始讨论B题，求数学期望，从这里就开始卡了，折腾了好久也没思路。
这时候已经过去三个多小时了，我们才出了两个题。我开始读其他题，G题求安排比赛方法，应该是排列组合问题，写出来超限，不出正确答案，而且超时，调了半天也不行。然后我开始看J题，开始感觉没思路，然后一直翻其它题，卢景月开始解A题积分求面积。这个时候已经有队伍ak了。然后我静下来想J题，整了半天感觉有了思路，一个结构体排序，从中间元素左右调整，然后我就开始写，这时候卢景月还在调A题，不断发现错误，然后我们俩轮流用机器，打印了好多张代码。我在一边调J题，卢不断发现A的问题，然后我们还是轮流用电脑，他改一下发现不对，然后打代码、然后我接着调、打代码、换人。。。就这么轮流，最后我写出了J，怕超时，交了试试不对，然后在下面各种改，自己想的数也过了，交了3次还是不行。A题也是感觉没什么错误了，交了六遍还是不过。这个时候只剩半个多小时了，我们开始休息吃东西。然后我们就在看board，心想这下丢人了。闲聊了一会，卢想起了A题Tx和Px重合的情况没判断，然后加了几行代码，交了就过了。我们开始激动了，距离结束只剩10分钟了。我想再试试，就静下来看了看J题，猛然发现多组测试数据没加，于是改了一下交了，又对了，我们当时的心情真是难以形容，就因为这两个低级问题卡了这么长时间，不过有一种绝杀的感觉...然后我们开始改D题，最后也没对。就这样比赛结束了。
感觉此次比赛题目没有想象的那么难，山大的两个队伍ak。我们做的也还可以，没太大遗憾，感谢两个给力的队友。只是深感水平太低，回去要继续学算法练题。
OUC本次一金两银，星号队也都进了铜牌，成绩很好，但是山大确实强，两队ak，星号队都出8题，差距还是很明显的。
强大的OUCacm，加油！
