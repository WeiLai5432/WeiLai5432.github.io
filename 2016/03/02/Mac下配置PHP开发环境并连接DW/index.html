Mac下配置PHP开发环境并连接DW

发表于 2016-03-02 | | 阅读次数

(这篇文章第一次加目录，不知为啥有点乱。)

今天想学一下PHP，于是开始配置开发环境。

Mac OS 里自带PHP和Apache，需要配置一下。
Apache是服务器软件，用于运行写成的PHP文件。
另外网上的教程中都提到了要安装 MySQL，顺便安上了。

下载安装MySQL:

在MySQL官网下载即可：http://dev.mysql.com/downloads/mysql/
安装很简单，而且新版进行了简化，把原来的三个文件弄成了一个，直接安装好就会在偏好设置里出现启动服务的选项。

如果需要可以顺便安装MySQL Workbench(GUI Tool)。

启动 MySQL 可以直接在 System Preference 里开启服务，也可以使用终端命令，注意要使用终端命令登陆需要使用绝对路径 /usr/local/mysql/bin/mysql，可以添加别名或者添加路径或者存放一个指向/tmp/mysql.sock文件的软链接。
添加别名：

alias mysql=/usr/local/mysql/bin/mysql

alias mysqladmin=/usr/local/mysql/bin/mysqladmin

据说这个方法重启终端无效

添加路径：

PATH=”$PATH”:/usr/local/mysql/bin

添加软链接：

sudo ln -s /tmp/mysql.sock /var/mysql/mysql.sock

PHP 和 Apache：

先介绍命令行的方法，再介绍用MAMP的方法：

用命令行启动和关闭 Apache：

先启动Apache服务吧，然后在浏览器中输入localhost。出现It works字样，说明Apache就搞定了。

在Macbook pro下，Apache的网站服务器根目录在/Library/WebServer/Documents路径下。

PHP配置：

只需要修改 Apache 的配置文件添加对 PHP 的支持即可。
进入/etc/apache2/文件夹，找到 http.conf 文件，把里面这句话取消注释即可：

LoadModule php5_module libexec/apache2/libphp5.so

配置完后可以写一个简单的 PHP 文件测试，Apache的网站服务器根目录在/Library/WebServer/Documents路径下，在此写了 PHP 文件后，访问localhost/相应文件名.php即可看到。

用 MAMP 配置：

下载安装 MAMP ，打开之后选择 Launch MAMP，MAMP pro是收费的。
然后他会自动出来欢迎页面，默认的URL会以 http://localhost:8888 开头。
单击【Preferences】，从打开的面板顶部选择【Ports】。
面板显示 Apache 和 MySQL 分别运行的端口。
单击【Set Web & MySQL Ports to 80 & 3306】。
数字更改为标准端口：Apache 为 80，MySQL 为 3306。
单击【OK】，根据提示键入你的 Mac 密码。
MAMP 重新启动两台服务器。

打开 MAMP 控制面板，点击【Star Servers】
Apache Server 和 MySQL Server 旁的绿灯表明两台服务器都在运行。
此时，MAMP 欢迎页面将重新载入浏览器中。

这次，URL 中【localhost】后已没有了【 :8888】，也没有【:80】
因为端口 80 是默认值，添加【 :80】 没有必要，所以不添加它也没关系。
使用非标准端口时，只需要在冒号后跟数字即可。

为 DW 定义 PHP 节点：

这一步骤是在DW中添加配置好的服务器。

1. 找到文档根，

文档根是服务器查找网页的顶级文件夹。

它的位置取决于 PHP 测试环境的安装位置和方式。

以下是一般 PHP 设置的默认位置：

XAMPP： 【C:\xampp\htdocs】
IIS：【 C:\inetpub\wwwroot】
WampServer：【 C:\wamp\www】
MAMP：【 /Applications/MAMP/htdocs】
文档根的 URL 为【 http://localhost】。

注意：如果要使用 MAMP 默认端口，请使用 【http://localhost:8888】 作为 URL。

2. 指定工作文件夹

要指定工作文件夹：
在 Dreamweaver 中，选择【站点】>【新建站点】。
在【站点名称】文本框中，键入站点名称，如 【PHP Test Site】。该名称用于在 Dreamweaver 中识别此项目。
单击【本地站点文件夹】文本框旁的文件夹图标，浏览到测试服务器的文档根。
创建一个名为 【php_test】的文档根的子文件夹并选择它，然后单击【保存】。
（【本地站点文件夹】的位置取决于 PHP 的安装方式。）

3. 为 Dreamweaver 指定测试服务器

要指定测试服务器：
从【站点设置】对话框左侧的列中选择【服务器】。
单击对话框左下角的加号 (+) 按钮，添加一台新服务器。
在打开的对话框中，确保选择【基本】按钮，然后从【连接使用】列表中选择【本地/网络】。
为服务器命名，如 【Local Testing】。
将【服务器文件夹】设置为指定为【本地站点文件夹】的文件夹；在本例中， php_test （在服务器的文档根中）。
对于【Web URL】，键入要用于到达 php_test 文件夹（在服务器的文档根中）的 URL。因为文档根等同于http://localhost/ ，您需要在【Web URL】文本字段中键入 http://localhost/php_test/
注意：如果要使用 MAMP 默认端口，【Web URL】应为 http://localhost:8888/php_test/。
选择对话框顶部的【高级】按钮。它将显示远程服务器和测试服务器选项。
对于【服务器模型】（在【测试服务器】部分中），选择【PHP MySQL】
单击【保存】。
这是一台测试服务器，所以务必选中【站点设置】对话框中的【测试】复选框。根据需要，取消选中【远程】复选框。
单击【保存】，保存所作更改。

完成以上步骤即可进行测试，写好 PHP 文件后点击“实时视图”按钮即可看到效果。
