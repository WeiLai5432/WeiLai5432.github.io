MathJax-在网页或MarkDown中插入数学公式

发表于 2017-01-11 | | 阅读次数

前几天写博客遇到了要在文章中插入数学公式的问题，MarkDown 本身好像并不怎么支持数学公式，想起来 LaTeX 对数学和排版支持很好。于是就上网查了一下，发现了 MathJax 这个神奇的东西。
MathJax 是一个 JavaScript 库，是可以跨浏览器渲染数学公式的引擎。支持 LaTeX, MathML 和 AsciiMath 标记。并且是开源的。

MathJax is a cross-browser JavaScript library that displays mathematical notation in web browsers, using MathML, LaTeX and ASCIIMathML markup. MathJax is released as open-source software under the Apache License.
