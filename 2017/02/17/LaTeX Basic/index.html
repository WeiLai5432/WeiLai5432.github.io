LaTeX 基本用法

发表于 2017-02-17 | | 阅读次数

很早就接触安装了 LaTeX，感觉很强大，但一直没完整地写一篇 LaTeX 文档，还是 MarkDown 比较轻便。最近用LaTeX写了综述，学到了很多东西，总结一下。

MacTeX 官网上有两个方案可以选择，一个是下载 The Full MacTeX 大礼包（2G+），另一个是下载Basic版本（几百M），Basic 只有必须的编译环境和基本的包，对于新手或者不想折腾的同学还是推荐下载完整的软件包。

完整的 The Full MacTeX 大礼包里面有自带的 TexShop 编辑器，试了一下也挺好用。我用的编辑器是 Sublime Text 3.

关于 LaTeX 的安装和中文环境的部署，网上有许多相关教程，我也转载过一篇，推荐这篇。

简要介绍一下TeX语法，我们写的TeX代码有很多代码是控制格式用的，好像叫做控制序列，以\符号开头，此外\符号还用作一些特殊符号的转义。因此文档的内容有些是显示出来的，有些是用来控制格式或生成图表一类的。

一些常用的控制命令和符号：

  • 取消缩进：\noindent{}

  • 空格：~

  • 换行：\\

  • 段落结束：\par

  • 代码注释：%

  • 加粗：\textbf{加粗的内容}

  • 项目符号：

  • 引用文献：\cite{}

  • 添加 label（用于引用此处）：\label{123}

  • 引用文档内的 label（自己添加的）：\ref{123}

  • 居中：\centering
