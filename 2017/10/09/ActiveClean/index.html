ActiveClean

发表于 2017-10-09 | | 阅读次数

最近读了 ActiveClean 的论文 ActiveClean: Interactive Data Cleaning For Statistical Modeling，记录一下。

ActiveClean 是一种交互式的关于统计模型的数据清洗方案。

分析人员通常迭代式地进行数据清洗，然后根据清洗结果处理更多数据。ActiveClean 就是这样的迭代式的框架，同时，ActiveClean 保证了模型训练的收敛性。

INTRODUCTION

脏数据会影响到模型训练的结果，而迭代式的数据训练也可能导致训练偏差甚至是在最简单的模型上出现很大误差。

To summarize our contributions:

• We propose ActiveClean, which allows for progressive data cleaning and statistical model training with guarantees.

• Correctness We show how to update a dirty model given newly cleaned data. This update converges monotonically in 1 expectation with a with rate O( 1/√T ).

• Optimizations We derive a theoretically optimal sampling distribution that minimizes the update error and an approximation to estimate the theoretical optimum. Our proposed optimizations can improve model accuracy by up-to 2.5x for the same amount of data cleaned.

• Experiments The experiments evaluate ActiveClean on ﬁve datasets with real and synthetic corruption. In a fraud prediction example, ActiveClean examines nearly 10x fewer records than alternatives to achieve an 80% true positive rate.

我们的目标是根据从原始数据（含脏数据）上训练得到的模型$\theta$得到一个合理的新的 $\theta$，如图：

最优的解即为图上的星号部分，也就是：（有时加正则化项r）
$$ \theta^* = arg\ min_\theta \sum_{i=1}^N \phi(x_i,y_i,\theta) + r(\theta) $$

$ \phi $ 的定义就是机器学习里的 Cost 函数：
$$ \phi(x_i,y_i,\theta) = ||\theta^Tx_i-y_i||^2_2 $$
