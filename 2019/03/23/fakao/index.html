法考备考

发表于 2019-03-23 | | 阅读次数

分享一点经验教训。

法考即国家统一法律职业资格考试（原国家司法考试），从2018年开始实行，与前司法考试相比考试形式和要求有很多变化。往年司考的已过时的内容就不提了。
通过此考试是取得法律职业资格证的途径，取得该证书的人可以依法担任执业律师、法官、检察官和公证员。
具体可以看历年司法考试公告和2018年首届法考的公告：2018公告
一个比较受关注的变化是从此年开始仅限制法律相关专业的人报考，但是公告公布前已取得学籍的人都可以报考，即“新人新办法，老人老办法”，法本的专业限制针对2018年及之后入学的人，此前满足司考报名条件的人同样可以报考法考。

考试报名时间：一般在每年6月中下旬，关注司法部的官方公告，固定在考前三个月发布。
考试时间：客观题一般在9月中下旬，主观题在客观题考试后一个月左右即10月中下旬，成绩公布一般在11月底。

考试形式：
客观题：两张卷，各一百道选择题，50单选，40多选，10不定项选，每张卷总分150分，合计300分，考试时间每张卷3小时。2018年全国已普及客观题机考，以后也会是机考。机考和纸质试卷的不同是题目顺序是打乱的，不像以前是某个部门法的题目是集中在一起。而且每个人的试卷题目顺序都不同。2018年客观题分数线是180分（估计以后的分数线也会固定到180分），通过客观题才能报考主观题。客观题成绩两年内有效，也就是说，如果本次只通过客观题未通过主观题，第二年可以直接报考主观题。
主观题：问答题一张卷，总分180分。考试时间4小时。考试范围比客观题稍微小一点，但是更有深度考察更细。2018年分数线108分，（估计以后也会固定108分）。开卷考试，考试时配发一本法律汇编的书可以查法条。2018年大部分地区是笔写，部分地区试行计算机打字，我估计2019年也会普及计算机考试。
通过客观题和主观题，即通过考试。可以按要求申领证书。

考试内容：按照部门法可分为八个部门：民法、刑法、民诉法（民事诉讼法（含仲裁制度））、刑事诉讼法、行政法（行政法与行政诉讼法）、理论法（中国特色社会主义法治理论、法理学、宪法、中国法律史、司法制度和法律职业道德）、商经法（知识产权法、商法、经济法、环境资源法、劳动与社会保障法）、三国法（国际法，国际私法、国际经济法）

下面讲如何备考。

主要是听课+看书+做题，目前很多法考辅导机构都有在线的免费讲课视频可以看，也有APP可以在线听课+在线做题，所以我觉得基本不用报班，只要花钱买书即可。比较有名的机构有厚大和瑞达，其他也可以选，选哪家不重要，最重要的是选的老师适合自己。

一般备考的课程会把备考时间分成几个阶段：
系统强化：即理论精讲，这个阶段是最重要的，这部分已经把所有的考试内容讲完了（法律法规有变化的除外）。时间上零基础听大概需要3个月。按照正常的节奏应该是3月-7月听这部分。
题目讲解：这部分老师会带着做题讲题。我没听过，题目是自己做看答案的。时间上，做题和听前部分的课可以穿插进行。
精讲阶段：和强化阶段一样是讲授，不会特别细，精简了一部分内容，但是会补充一些内容，如果时间充分要听一下，帮助复习抓一些重点，也能掌握法律变化的东西。时间上，在6-9月听。
最后会有小部分真题讲解。
主观题的一个月会另外录制一部分课程，都是在讲做问答题。

时间安排要看自己的时间和精力，一般的节奏是3月开始，6月之前的时间是在听系统强化课穿插做题，6-9月就是听精讲课程和做题，到9月考试时差不多可以学完系统强化+精讲，做完五年的真题加上几百道模拟题，基本就够了。如果学得节奏快可以适当缩短时间。
去年我是从6月才开始准备，零基础已经很晚了，只听了强化部分的课（一般是两三倍速听），中间穿插自己做题，导致考前两天还在听课，差点没听完课，所以基础就不牢。

很重要的一方面是选择老师，每个阶段每个部门法都有不同的老师在讲，选适合自己的老师很重要。当时我的书是别人的一整套，加上六月才开始准备，所以没有时间精力挑，就按照教材选了对应的厚大的老师。如果有时间精力尽量先去几个机构上听一下老师的课，然后再选择买书和听课。

书的方面，针对上面几个阶段，机构会推出相应的教材。有学习包形式（即一整套教材），也会卖单科的对应老师编的教材，有精力自己选一些适合自己的老师买对应的书比较好，买学习包整套也可以。如果要买学习包，尽量看一下这些老师是不是自己喜欢的。我推荐是分开单科买。不同的学科选不同的机构不同的老师都可以，但是单科选择老师的课程和选书应该是同步的，不然会有书和课程对不上的情况。
我的经验是买理论卷+真题就够了，精讲阶段会有比理论卷薄一点的教材，有精力可以买。注意，如果想省心的话，尽量买新的教材，不要用往年的，因为每年法律内容都有变化，用往年的书会有一些过时的东西。我的书是别人送的17年的，用来复习18年考试就有很多变动的地方，用起来有很多要自己注意。

结合考的部门法，推荐一些我听过的老师：
民刑（民法+刑法+民诉+刑诉）是法考最重要的部分，占据60%的内容。
民法我听过张翔老师和段波老师的课，段波去年离开后大和其他老师自己搞了个机构，这两个老师感觉都可以选，我后来听了段波感觉适合我一点。
刑法我听了张宇琛和罗翔老师的课，推荐罗翔，中国政法大学教授，我觉得讲得很好，不推荐张宇琛。另外瑞达也有有名的老师比如刘凤科，我没听过，有精力可以自己听听看。
民诉法听了刘鹏飞，推荐，爱讲段子，幽默，讲课也挺好。别的老师我没听过。
刑诉法我听了向高甲，非常推荐。幽默，讲得清晰，还有很多帮助记忆的口诀。
行政法只听了魏建新，感觉一般，但是也没听过别的老师，可以自己选一下。
商经法只听了鄢梦萱，还行把，可以听听别的老师。
三国法，殷敏，还可以，条理清晰，人很亲切，别人的课我没听过。
理论法，听过白斌和高晖云，还有瑞达的宋光明，推荐高晖云和宋光明，不推荐白斌，可能是我加速的原因感觉他讲话节奏有点怪，而且声音不稳。
学习顺序，我当时零基础什么也不懂，就按照教材编号听课，但是这样有很多问题，比如行政法和民诉法有很多关联，商法和民法也有一些关联，所以这个顺序比较不科学。
我的建议是先听理论法中法理学的部分，或者先学理论法也可以，然后民法-民诉，刑法-刑诉，行政法，理论法，商经法，三国法。
